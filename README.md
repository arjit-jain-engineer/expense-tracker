# Paisa — Expense Tracker with Friend Settlements

Paisa is a full-stack expense tracker for managing personal expenses, splitting bills with friends, and tracking settlements. It uses a static frontend with HTML, Tailwind CSS, and JavaScript, while Supabase provides authentication, database, and secure APIs with Row Level Security, and Vercel handles deployment.[1][2][3]

## Tech Stack

- HTML
- Tailwind CSS
- Vanilla JavaScript
- Supabase
- Vercel

## What it does

- Track personal expenses
- Create groups
- Split expenses with friends
- See who owes whom
- Record settlements

## Architecture

- Frontend: HTML, Tailwind CSS, JavaScript
- Backend: Supabase Auth, PostgreSQL, REST API, RLS
- Deployment: Vercel

## Note

This project is built as a simple full-stack web app without a separate custom backend server. Supabase auto-generates APIs from the database schema and uses Row Level Security for data protection.[4][5]
